HTML Tag Reference

<head> Tag: The container for all the head elements (e.g. the title of the tab)

<header> Tag: The headers included within the body of the webpage, the "HTML Tag Reference" at the top is an example.

<body> Tag: Defines the document's body,all the information you see on a webpage is in the body

<p> Tag: Paragraph tag defines....well.. a paragraph within the body

<a> Tag: The anchor creates hyperlinks within the body that leads to another page

<div> Tag: Divides the webpage into sections that can be accessed with anchors. E.g. instead of scrolling down all the way to an author's contact information on an article, you can click a piece of text that has a div tag, and it will send you to the bottom of the page

<hr> Tag: Creates a horizontal rule in a webpage

<style> Tag: The styletag is used to define style information for an HTML document.Inside the style element you specify how HTML elements should render in a browser.

<img> Tag: Places an image within the webpage using the link address

<button> Tag: Inside a button element you can put content, like text or images.

<video> Tag: Like the img tag, the video tag places a video using the link address

<u> Tag: Underlines and specifies text in a webpage

<center> Tag: Center tag is used to center align text instead of being on the side of a webpage, like the heading of this page

<strong> Tag: Bolds specific text in a webpage

<font> Tag: Changes font that is in a webpage

<ul> Tag: Creates a bulleted list

<br> Tag: Creates a line break within the webpage

<abbr> Tag: Defines abbreviations within the text

<audio> Tag: Like the img and video tags, video tags place audio within a webpage using the link address

<table> Tag: Creates a table within a webpage
